5
English

Commands

  • Used to spawn entities
  • Acts like a command buffer

Assets<Mesh>

  • Stores all mesh data globally
  • Mesh = geometry

  • Stores surface appearance
  • Material = color + lighting response

Assets<StandardMaterial>
